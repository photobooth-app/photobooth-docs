# Customize the Theme

To customize the visual apperiance of the photobooth app, add your rules to the `./userdata/private.css` file.

## Example: Adjust the Buttons' Size on the Frontpage

```css
.action-button {
    font-size: 1.0rem;
}
.action-button *>.q-icon {
    font-size: 4.5rem;
}
```
